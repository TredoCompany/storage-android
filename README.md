# storage-android
Модуль c утилитами для SharedPreferences, DataStore, KeyStore

## **Использование**

build.gradle(root)
```groovy
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}
```

build.gradle(app)
```groovy
dependencies {
    implementation 'com.github.TredoCompany:storage-android:1.0.0'
}
```
